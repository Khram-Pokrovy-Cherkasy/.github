# Contributing to Pokrova Open Liturgy

We welcome contributions from developers, linguists, and the theological community!

### How to contribute:
1. **Development:** Help us with OSIS XML parsing or Neo4j data modeling.
2. **Content:** Review and verify digitized liturgical sequences.
3. **Documentation:** Improve technical or user guides.

Please open an **Issue** before submitting a significant Pull Request.
