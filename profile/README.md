# Khram Pokrovy Cherkasy — Pokrova Open Liturgy (POL) ⛪💻

## 🌟 Mission & Project Overview
**The first community of the Ukrainian Autocephalous Orthodox Church in Cherkasy since Independence. Currently part of the Orthodox Church of Ukraine (OCU).**

We bridge rich spiritual heritage with modern technology. Our flagship project, **Pokrova Open Liturgy (POL)**, focuses on the digitalization and systematization of Ukrainian liturgical texts, making them accessible, searchable, and machine-readable.

### Technical Goals:
- **Digitalization:** Transforming Biblical and Liturgical texts (Typikon) into a high-performance **Neo4j Graph Database**.
- **Accessibility:** Developing open-source tools for OCU communities and theological research.
- **Infrastructure:** Scalable deployment utilizing **Microsoft Azure** (Nonprofit Grant) and Docker orchestration.

## Про громаду та проект
**Перша громада Української Автокефальної Православної Церкви в м. Черкаси часів Незалежності. Нині Православна Церква України.**

Ми поєднуємо духовну спадщину з технологіями. Проект **Pokrova Open Liturgy** спрямований на оцифрування українських літургійних текстів.

### Наші цілі:
- **Цифровізація:** Перетворення текстів Біблії та Уставу у формат графової бази даних (Neo4j).
- **Доступність:** Створення відкритих інструментів для громад ПЦУ.
- **Інфраструктура:** Розгортання в хмарі Microsoft Azure за підтримки некомерційного гранту.
